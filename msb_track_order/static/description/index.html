¿Como agregar mensajes?

Abrir la SO y agregar un comentario de la siguiente manera:
t-name Venta confirmada t-name

t-desc la orden fue confirmada t-desc

¿De donde se obtiene Fecha estimada de entrega?

Es necesario tener configurado lo siguiente

Ventas/Configuración/Ajuste/Ventas/Envíos/Fechas de entrega. De la Cotización/Pedido de venta/Fecha de de compromiso

¿De donde se obtiene código de rastreo ?

Es necesario tener instalado el modulo de inventario.
